House Prices - Advanced Regression Techniques.

## კონკურსის მიმოხილვა 
House Prices კონკურსის მიზანია თითოეული სახლის ფასის პროგნოზირება. მოცემული თვისებების საფუძველზე უნდა ვიწინასწარმეტყველოთ ყოველი მოცემული Id-სთვის შესაბამისი ფასი. ეს ამოცანა ფასდება საშუალო კვადრატული ცდომილებით(RMSE).
---

### მონაცემთა წაკითხვა და სწორად დასპლიტვა
ჩვენ ვიყენებთ supervised learning-ს შესაბამისად სანამ რაიმე მონაცემის გავფილტრავდეთ უნდა დავსპლიტოთ ეს მონაცემი. გვაქვს input და output, შესაბამისად ცალკე უნდა გამოვყოთ 'SalesPrice' და ისიც იგივენაირად უნდა დავყოთ და იგივე მახინაციები ჩავატაროთ, რასაც ვიზამთ input-ის train და validation ნაწილებზე. ვინაიდან ვიყენებთ cross-validation უნდა დავსპლიტოთ.

### 2. NA შევსენა
  სანამ უშუალოდ შევსებაზე გადავალთ დაჭიროა კატეგორიული ცვადების და რიცხვით ცვადების გამოყოფა, რომ სხვანაირად შევავსოთ

  თავიდან რიცხვითი ცვლადები მოდი შევავსოთ საშუალოთი, ხოლო კატეგორიული ცვლადები შევავსოთ მოდით ან შეგვეძლო 'None' მაგრამ მე პირადად ვამჯობინებ მოდას. 



ვინაიდან გვაქვს supervised learning გვაქვს input და output შესაბამისად სანამ დავსპლიტავთ ტრეინგ და ტესტ-ებად უნდა გამოვყოთ 'target', ანუ უნდ აგამოვყოთ 'SalesPrice' .
ახლა ვნახოთ ნალები, ისეთი სვეტები, რომელთა უმრავლეობა არის ნალი შეგვიძლია პირდაპირ დავიკიდოთ, პირდაპირ გადავაგდოთ, თუ ნახევარი მაინც არის ნალი მაშინ უნდა შევავსოთ ეს ნალი რაღაცით, საშულოთი ან მედიანით... ჩვენ შემთხვევაში შეგვილია პირდაპირ გადავაგდპთ სამი სვეტი, კერძოდ, 'PoolQC', 'MiscFeature', 'Alley', რადგან მათი თითიქმის ყველა შედეგი არის ნალი , დანარჩენები კი შევავსოთ საშუალოთი.
